# nasij_monorepo_v2_2_fabric_dashboard — IoT + لوحات Grafana

> **المدينة تنبض. حسّاسات حيّة تتدفَّق إلى DB، تتجمَّع في لوحة.**

## الميزات المُضافة على v2.1

### بنية IoT كاملة
- **Mosquitto MQTT broker** على :1883.
- **iot-listener service** يشترك في `sensors/+/+`.
- جدول `iot_events(topic, payload, ts, geom)`.
- Prometheus counter `iot_events_total`.

### لوحة Grafana
- `infra/grafana/dashboards/urban.json` — ETL request rate + p90 latency.

### نقاط نهاية جديدة
- `GET /kpi/summary` — Fabric Index, Sprawl Risk, Harmony.
- `GET /growth/hotspots` — مراكز الخلايا منخفضة الترابُط.
- `GET /ogc/collections` — OGC Features API.

### Patch مرفق
`nasij_v2_2_to_v2_3.patch` — تعليمات الانتقال إلى v2.3.

## ثغرات حرجة

- 🔴 MQTT بلا TLS + كلمات مرور افتراضيّة `nasij/nasij`.
- 🔴 لا حدّ لحجم MQTT payload (DoS).
- 🔴 lateral exception handling في listener.py (`except Exception: pass`).
- 🟠 lots of magic numbers في الـ KPI formulas.

→ النسخة التالية: [`v2.3`](v2_3.md) تُضيف H3 + Gi*.
