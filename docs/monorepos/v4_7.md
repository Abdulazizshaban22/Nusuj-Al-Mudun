# nasij_v4_7_monorepo — معايير OGC + STAC + CQL2

> **النسخة الأكثر التزامًا بالمعايير المفتوحة. تحوُّل من NestJS إلى Express لـ flexibility.**

## التحوُّلات الجوهريّة

### من NestJS إلى Express
- إعادة كتابة الـ API بـ Express 4.19 + body-parser + jsonwebtoken + jwks-rsa.
- TypeScript `strict: false` (انحدار من v4).
- في-memory feature stores (لا DB) — للـ demo فقط.

### إضافة OGC API Features
- `GET /ogc/collections` — قائمة المجموعات.
- `GET /ogc/collections/features/items?bbox=...&datetime=...&filter=...` — استرجاع.
- `GET /ogc/collections/open311_requests/items` — بلاغات.
- `POST /ogc/query/spatial` — within, intersects, touches, disjoint, overlaps, contains.

### إضافة STAC 1.0.0
- `GET /stac` — Root catalog.
- `GET /stac/collections` — Collections.
- `POST /stac/search` — بحث بـ collections + datetime + bbox.
- مثال Item: `ndvi-2025-10.json` على S3 (COG).

### CQL2 Support
- CQL2-text: `?filter=status%3Dopen%20AND%20priority%3E%3D3`.
- CQL2-JSON: nested logical expressions.

### JWT Authentication
- `VERIFY_JWT` env var (افتراضي **false**).
- `jwks-rsa` يربط Keycloak JWKS endpoint.

### Removed
- PostgreSQL (تمامًا).
- Mosquitto.
- Grafana + Prometheus.
- Web frontend.
- Database init scripts.

## ETL Service مُبسَّط
- Flask بدلاً من FastAPI.
- `numpy` فقط (لا pandas، لا scikit-learn).
- `/health`، `/alerts/events/add`، `/st/gi_star` (stub).

## ثغرات

- 🔴 **`VERIFY_JWT=false`** افتراضيًا — يجب enforce في prod.
- 🔴 BBox validation ضعيف (لا تحقُّق من min<max).
- 🔴 In-memory store غير محدود الحجم — DoS عبر `/geojson/ingest`.
- 🟠 parseCQL2 يقبل أسماء حقول بنقاط.
- 🟠 `inDatetimeRange` يرجع `true` عند خطأ تحليل التاريخ.
- 🟠 18+ instances من `any` في TypeScript.

## استخدام مقترَح

نموذج لتطبيق OGC/STAC. **لا persistence**، لذا للاختبار فقط.
