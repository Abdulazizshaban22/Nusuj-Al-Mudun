# nasij_monorepo_v1_1 — الخوارزميّات والاستشعار

> **بعد إثبات العمارة، نُضيف الذكاء التحليلي.**

## الميزات المُضافة على v1
- **AHP** كامل مع نسبة الاتساق (CR).
- **TOPSIS** لترتيب البدائل.
- **DBSCAN** للعنقدة المكانيّة.
- **Isolation Forest** لكشف الشذوذ.
- **مؤشِّرات استشعار عن بُعد:** NDVI و LST_BT.
- وثائق `POSTGIS_LAYERS.md` و `OSRM.md` (حُذِفت، أُعيدت إلى Reference).

## الـ API الجديدة

| Endpoint | Method | الوصف |
|---|---|---|
| `/algo/ahp` | POST | حساب أوزان |
| `/algo/topsis` | POST | ترتيب بدائل |
| `/algo/dbscan` | POST | عنقدة بكثافة |
| `/algo/iforest` | POST | كشف outliers |
| `/rs/ndvi` | POST | NDVI من NIR/RED |
| `/rs/lst_bt` | POST | LST من الباند الحراري |

## ما زال غير مكتمل

- الواجهة لا تستفيد من الخوارزميّات بعد.
- لا تخزين للنتائج في DB.

→ النسخة التالية: [`v1.2`](v1_2.md).
